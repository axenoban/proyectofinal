¿Qué hace cada cosa?

index.php — Archivo principal, punto de entrada. Aquí decides qué vista mostrar según lo que venga por URL o formulario.

controllers/ — Aquí la lógica para procesar datos, conectar con modelos y elegir qué vista cargar.

models/ — Código para la base de datos (consultas SQL, conexión, etc.).

views/ — Archivos HTML/PHP que se muestran al usuario (formulario login, lista de pedidos, etc.).

assets/ — Bootstrap, JS, imágenes, estilos.
